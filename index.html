About me

I'm an enginering student following a degree in Data Interaction and student programmer at EasyTranslate (Copenhaguen), I create new systems using machine learning for the translation industry. I'm interested in technology, science and innovation, particularly concerning AI and space exploration. I like photography, traveling, photography and cats.

Projects

AR Pool

October 2014 - January 2015

One of my first computer sciences project. The project aims to help the pool players using augmented reality.

HeavyM

March 2015 - August 2015

HeavyM is an intuitive projection mapping software, it is currently running in beta version. It allows the creation and projection of visual animations with great facility, in order to be projected on real surfaces with a projector.

As an intern, my role was to developp from scratch the new version of the software announced during the Kickstarter campagne.

DeepReader

October 2015 - December 2015

This project involves natural language processing and network science.Its development is part of our MSc Digital Media Engineering at DTU and is through collaboration with EasyTranslate.

The project aims to provide a tool to understand a text without having to read it. We make it possible to explore a corpus of texts and display the relations between them.

TriangulateJS

March 2016

A short personal project made during holidays. This app generates automatically a polygon art from any image using Delaunay triangulation and contour detection. The app was originally coded with Python, but it is now running with Javascript.

Improving Shelter Animal Outcomes

Mai 2016 - June 2016

My first Kaggle competiton (a data mining competition).

Given a data set containing information about hosted animals (age, sex, color, breed, etc..) and their outcomes (Adopted, euthanasied, return to owner, etc..), the goal was to predict the outcomes of new animals given the information about animals.

Companion Light

A project that aims to build a context aware light using RGB LEDs, a BLE arduino module and an Android app. The light adapt itself depending on the position of the user.
