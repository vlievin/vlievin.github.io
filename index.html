Projects

HeavyM

HeavyM is an intuitive projection mapping software, it is currently running in beta version. It allows the creation and projection of visual animations with great facility, in order to be projected on real surfaces with a projector.

As an intern, my role was to developp from scratch the new version of the software announced during the Kickstarter campagne.

DeepReader

This project involves natural language processing and network science.Its development is part of our MSc Digital Media Engineering at DTU and is through collaboration with EasyTranslate.

The main goal here is to provide all the tools to understand a text without having to read it. We make it possible to explore a corpus of texts and display the relations between them.

TriangulateJS

A short personal project made during holidays. This app generates automatically a polygon art from any image using Delaunay triangulation and contour detection. The app was originally coded with Python, but it is now running with Javascript.
