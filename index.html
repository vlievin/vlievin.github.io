Projects

Most of my projects are available on my Github. I've been contributing to projects related to Data Science, Android development, Software development and Augmented Reality as well as simple web apps. Recently, I've been focusing on projects involving Deep Learning and Reinforcement Learning. Here are some of my favorite projects

HeavyM

March 2015 - August 2015

HeavyM is an intuitive software which aims to bring projection mapping to the crowd. It allows the creation and projection of visual animations with great facility, in order to be projected on real surfaces with a projector.

As an intern, my role was to developp from scratch the new version of the software announced during the Kickstarter campagne.

DeepReader

October 2015 - December 2015

This project involves natural language processing and network science. Its development is part of our MSc Digital Media Engineering at DTU and is through collaboration with EasyTranslate.

The project aims to provide a tool to understand a text without having to read it. We make it possible to explore a corpus of texts and display the relations between them.

Companion Light

Mai 2016 - June 2016

A project that aims to build a context aware light using RGB LEDs, a BLE arduino module and an Android app. The light adapt itself depending on the position of the user.
